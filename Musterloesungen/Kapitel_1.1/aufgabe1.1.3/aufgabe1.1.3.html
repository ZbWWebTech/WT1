Wichtiger Text oder wichtige Informationen können mit semantisch speziell ausgezeichnet werden. Auch wenn visuell kein Unterschied zum fett gedruckten Text zu erkennen ist, hat er doch eine andere semantische Bedeutung.

Text kann kleiner (small) ausgezeichnet werden.

Text kann eingefügt (inserted) ausgezeichnet werden.
Text kann tiefgestellt (subscript) ausgezeichnet werden.
Text kann hochgestellt (superscript) ausgezeichnet werden.
